# 修改字段（表结构）





> 

## 新增字段

ALTER  TABLE 表名

ADD 【COLUMN】 字段名 字段类型 【first | after 字段名】

> 添加单个字段，可选择位置

ADD(字段1 字段类型, 字段2 字段类型,……)

> 添加多个字段，向后依次添加

## 删除字段

ALTER TABLE 表名 DROP 字段名

# 数据操作

## 增

### 为所有字段添加数据

INSTER 【INTO】 表名 VALUE(数据1,数据2……);

### 为部分字段添加数据

INSTER 【INTO】 表名(字段1,字段2…)  VALUES(值1,值2)【,(值1,值2)】;

INSTER 【INTO】 表名 SET 字段1=值1【,字段2=值2……】;

### 一次添加多行数据

INSTER 【INTO】 表名【(字段1,字段2…)】 VALUES(数据1,数据2…),(),(),……;
